Executive SummaryIndia's electoral rolls are public. They contain each voter's name, age, address, booth assignment, and family relationship codes. Individually, every entry looks routine. In a graph, the anomalies scream: 50 voters sharing a single room, families where the son is older than the father, entire booths that appeared overnight, or one address that spans three different pin codes.The Ghost Voter Network Detector uses TigerGraph Savanna via Model Context Protocol (MCP) to ingest electoral roll data, model voter relationships as a connected graph, and run six coordinated detection algorithms in under one second per constituency query. The result: a ranked, evidence-backed list of ghost voter clusters, complete with PDF evidence packages ready for submission to the Election Commission of India (ECI).Why graph â€” not SQL?SQL can find 50 people at one address. Only a graph can find the coordinated multi-address, multi-booth network behind it â€” tracing the spider web of shared devices, impossible family trees, and overnight batch enrollments that prove systematic, organised fraud rather than data entry errors.2. find_duplicates(df):    # Exact EPIC duplicates    exact_dups = df[df.duplicated('epic_id', keep=False)]     # Phonetic near-duplicates: same soundex + DOB within 365 days    df['soundex'] = df['name'].apply(jellyfish.soundex)    groups = df.groupby('soundex')    near_dups = []    for _, grp in groups:        if len(grp) > 1:            for i, row_a in grp.iterrows():                for j, row_b in grp.iterrows():                    if i >= j: continue                    dob_gap = abs((row_a.dob - row_b.dob).days)                    lev = jellyfish.levenshtein_distance(row_a.name, row_b.name)                    if dob_gap <= 365 and lev <= 2:                        near_dups.append({                            'voter_a': row_a.voter_id,                            'voter_b': row_b.voter_id,                            'similarity': 1 - (lev / max(len(row_a.name), len(row_b.name))),                            'reason': 'PHONETIC_NAME_DOB_MATCH'                        })    return exact_dups, pd.DataFrame(near_dups)4.4 TigerGraph Loading via MCP# Load via MCP tool: loading_job_run# Or direct REST++ for high-volume batches import requests SAVANNA_URL =
